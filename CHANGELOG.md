# Changelog

All notable changes to TIC Coder Lite will be documented in this file.

## 0.0.1 - Demo Preview

### Added

- Lite Mode deterministic scanner.
- Stack and architecture detection by convention.
- Lightweight graph generation without Neo4j or a database.
- Deterministic risk detection.
- `.tic-code/` context files for AI agents.
- Standard AI exports for Codex, Claude Code, GitHub Copilot, Cursor, Gemini CLI, and Aider conventions.
- Optional Local AI Mode with Ollama.
- VS Code WebView overview with three visible modes.
- Output Channel logs, progress notifications, cancellation, scan limits, and incremental scan reuse.
- Demo-ready documentation, notices, debug tasks, and local VSIX packaging script.

### Notes

- This version is intended for local demos and manual `.vsix` testing.
- It is not published to the VS Code Marketplace.
